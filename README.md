## Trabalho do Professor Guilherme

Este projeto foi desenvolvido como atividade proposta pelo professor Guilherme, onde era necessário criar um site livre.  
Optei por desenvolver um site de uma academia.

---

## Sobre o projeto
O site apresenta uma academia com as seguintes funcionalidades:

- Catálogo de serviços  
- Planos da academia  
- Tela de login  
- Página inicial com apresentação  
- Informações gerais sobre a academia  

---

## Design
- Cor principal: Amarelo  
- Estilo moderno e chamativo  
- Foco em organização e experiência do usuário  

---

## Objetivo
Desenvolver um site funcional aplicando conceitos de desenvolvimento web.
